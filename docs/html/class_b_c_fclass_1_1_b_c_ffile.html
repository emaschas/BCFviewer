Detailed Description

Content of one or multiple BCF files. Contains a list of Topics in the property TopicsList

BCF : BIM Collaboration Format
See : https://technical.buildingsmart.org/standards/bcf/
and : https://github.com/buildingSMART/BCF-XML/tree/master/Documentation

There are two constructors :

  • BCFfile() that creates an empty object
  • BCFfile(FileName) that creates an object and reads the content of a BCF file.

The class provides one method :

  • ReadBCF() to read or append a BCF file into the object.

And one property :

  • TopicsList that references all the Topics that have been read and appended with the above methods.
